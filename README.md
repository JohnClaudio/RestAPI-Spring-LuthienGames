# Java Spring LuthienShop


## REST API Endpoints


```
/produtos

  GET / - LISTA DE TODOS PRODUTOS CADASTRADOS
  GET /id/{} - PESQUISA DE PRODUTO POR ID
  GET /nome/{} - FILTRO DE PESQUISA DE PRODUTOS PELO NOME
  GET /nome/ - FILTRO DE PESQUISA DE PRODUTOS PELO NOME
  GET /imagens/ - RETORNA IMAGENS DOS PRODUTOS CADASTRADOS
  POST / - ADICIONA NOVO PRODUTO NA BASE DE DADOS
  PUT / -  ATUALIZAÇÃO DE UM PRODUTO
  DELETE / -  REMOVE PRODUTO PELO ID
  
/categoria

  GET / - LISTA TODAS CATEGORIAS CADASTRADAS
  GET /id/{} - PESQUISA DE CATEGORIA POR ID
  GET /nome/{} - FILTRO DE CATEGORIA DE PRODUTOS PELO NOME
  GET /nome/ - FILTRO DE CATEGORIA DE PRODUTOS PELO NOME
  POST / - CRIAÇÃO DE NOVA CATEGORIA
  PUT / -  ATUALIZAÇÃO DE UMA CATEGORIA
  DELETE / -  REMOVE CATEGORIA PELO ID
